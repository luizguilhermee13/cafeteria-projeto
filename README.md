☕ Café Gourmet - Sistema de Cafeteria
Este projeto foi desenvolvido como Trabalho de Conclusão de Curso (TCC) no ano de 2024, para o curso superior de Sistemas de Informação. O objetivo principal foi a criação de uma plataforma web para uma cafeteria gourmet, integrando uma experiência visual agradável com funcionalidades essenciais para o gerenciamento de pedidos.

🚀 Sobre o Projeto
O sistema foi idealizado para atender às necessidades de uma cafeteria que busca digitalizar seu atendimento. Durante o desenvolvimento, o foco principal foi a construção de uma interface intuitiva e a implementação de regras de negócio fundamentais para o fluxo de compra.

Principais Funcionalidades:
Gestão de Carrinho: Implementação completa da lógica de adição, visualização e manipulação dos itens selecionados pelo cliente.

Interface Gourmet: Layout desenvolvido com foco na identidade visual de uma cafeteria de alto padrão.

🛠 Etapas de Desenvolvimento e Aperfeiçoamento
Para a conclusão do projeto, as seguintes etapas estão em fase de implementação e refinamento:

Implementação da Lógica do Carrinho: Desenvolvimento das funcionalidades de manipulação de itens, incluindo adição, remoção e cálculo de totais em tempo real.

Refatoração do Código: Otimização da base de código atual para garantir maior legibilidade, modularidade e melhor performance do sistema.

Aprimoramento de Front-end: Implementação de ajustes finos na interface para melhorar a interação do usuário.

Otimização de Responsividade: Ajustes nos estilos CSS para garantir que a cafeteria gourmet seja perfeitamente acessível e visualmente coerente em diferentes dispositivos (mobile, tablets e desktops).

🛠 Tecnologias Utilizadas
O projeto foi construído utilizando as tecnologias base da web, priorizando a simplicidade e a performance:

HTML5: Estruturação semântica de todas as páginas.

CSS3: Estilização responsiva e design focado na temática gourmet.

JavaScript: Implementação da lógica de interação, manipulação do DOM e gerenciamento do carrinho de compras.

GitHub: Controle de versão e hospedagem do repositório.
